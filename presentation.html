Chapter III

Cơ sở dữ liệu.

Sử dụng mô hình Document-based để tối đa hóa tính linh hoạt của cấu trúc sản phẩm đồ ăn vặt.

Chi tiết Product Schema.

const ProductSchema = new Schema({
  name: { type: String, required: true },
  price: { type: Number, required: true },
  collectionType: { type: String },
  images: [{ type: String }],
  inStock: { type: Boolean, default: true }
});

Chi tiết Order Schema.

const OrderSchema = new Schema({
  items: [{ productId: ObjectId, quantity: Number }],
  guestInfo: { name, email, phone, address },
  totalAmount: Number,
  status: { type: String, default: 'pending' },
  paymentMethod: { type: String, default: 'cod' }
});
